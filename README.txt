Overview ------------>

    Frontend interface for the Face Recognition Attendance System, built with React + Vite:

        - User-friendly Admin Dashboard & User Dashboard
        - Face capture via webcam for signup and attendance marking
        - Display attendance stats & analytics
        - Manage users and view attendance history

Tech Stack ------------->

    - React.js + Vite
    - Tailwind CSS (UI styling)
    - React Router (Routing & navigation)
    - Axios (API calls)
    - Face-api.js (Face detection & embeddings)
    - Webcam.js (Capture webcam images)

Quick Start -------------->

    1.Clone the repo:
        git clone https://github.com/smriti260402/faceAttendance-FE.git
        cd face-attendance-frontend
    2.Install dependencies:
        npm install
    3.Run the development server:
        npm run dev
    4.Open in browser:
        http://localhost:5173

Key Features ----------->

    - Signup / Login: Secure authentication with JWT
    - Face Capture: Capture 5 images for user registration
    - Attendance Marking: Real-time face verification & attendance logging
    - Admin Dashboard: Stats, user management, attendance history
    - User Dashboard: View personal attendance stats and mark daily attendance
    - Modal & UI animations: Interactive modals for face capture and history
